# 🚀 Portfolio — Bruno Castillo

Portafolio personal como **Full Stack Developer**, construido con React + TypeScript + Vite.

## 🔗 Demo en vivo
> Próximamente en [brunocastillo.dev](https://brunocastillo.dev)

## 🛠️ Stack
- **React 18** + **TypeScript**
- **Vite** — bundler ultrarrápido
- **CSS puro** — sin frameworks de estilos
- **GitHub API** — actividad de contribuciones en tiempo real

## 📁 Estructura del proyecto

```
src/
├── assets/
│   ├── fotos/          # Fotos personales
│   └── marcas/         # Logos SVG de marcas
├── components/
│   ├── Navbar.tsx
│   ├── Hero.tsx
│   ├── About.tsx
│   ├── Experience.tsx
│   ├── Skills.tsx
│   ├── Projects.tsx
│   ├── GitHubActivity.tsx
│   ├── Testimonials.tsx
│   └── Contact.tsx
├── data/
│   └── portfolio.ts    # Datos centralizados
└── styles/
    ├── index.css
    ├── extras.css
    └── exp-tabs.css
```

## ⚡ Instalación

```bash
# Clonar el repositorio
git clone https://github.com/BrunoCastillo23/portfolio.git

# Instalar dependencias
cd portfolio
npm install

# Modo desarrollo
npm run dev

# Build producción
npm run build
```
